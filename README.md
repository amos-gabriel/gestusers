# gesUsers - Système de Gestion des Utilisateurs

## 📋 Présentation
Ce projet est une application web de gestion d'utilisateurs développée dans le cadre du **Master 2 GLSI** à l'**Ecole Supérieure Polytechnique (ESP)**. 
L'application permet d'ajouter, de lister, de modifier et de supprimer des utilisateurs via une interface moderne et intuitive.

## 🛠️ Technologies utilisées
* **Langage :** Java (Jakarta EE)
* **Architecture :** MVC (Model-View-Controller)
* **Serveur d'application :** Apache Tomcat 11
* **Base de données :** PostgreSQL / PostGIS
* **Design :** CSS3 (Thème Bleu Professionnel) & JSTL
